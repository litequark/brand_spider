## 行政区划处理原则

由于各个品牌API提供的数据不尽相同，对于行政区划的处理，有以下原则：

- 只要API返回值中包含行政区划字段（如省、市、区），就把**API提供的字段**按原样写入CSV文件中；
- 对于不含在API返回值中的字段，写入CSV文件时**置空（仅写入空字符串）**。
